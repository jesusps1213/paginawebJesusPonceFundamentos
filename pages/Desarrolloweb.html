Desarrollo del Software

El desarrollo de software engloba el proceso de concepción, diseño, creación, pruebas y mantenimiento de sistemas informáticos. Todo este procedimiento se lleva a cabo mediante el código fuente, dedicado a la codificación, pruebas, despliegue y mantenimiento.

Repositorios

Los desarrolladores gestionan el código fuente a través de repositorios. Hay dos tipos principales:

  • Repositorio Central (VCS):
    • Contiene una copia oficial del código fuente.
    • Tipos: Subversión (SVN), Team Foundation Server (TFS).
  • Repositorio Distribuido:
    • No tiene un repositorio oficial y contiene una copia completa del código fuente.
    • Tipos: GIT, MERCURIAL.

GIT

GIT es un sistema de control de versiones distribuido y de código abierto. Sus características incluyen:

  • Control de versiones distribuido.
  • Los usuarios mantienen el código en máquinas locales.
  • Cambios locales pueden realizarse sin conexión a internet.
  • Sincronización de cambios locales mediante Pull & Push.

Comandos GIT

  • git clone: Crear una copia local de un repositorio remoto.
  • git init: Indicar la existencia de un nuevo repositorio GIT en el directorio local.

Además, GIT cuenta con README, un archivo común en proyectos de desarrollo web, utilizado para agregar información sobre la página.
